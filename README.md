# K72 React + GSAP

A frontend recreation inspired by the K72 marketing agency website, built with React, GSAP, Tailwind CSS, and Vite. The project focuses on smooth animations, responsive layouts, and a modern user interface while following a component-based architecture.

## Features

* Responsive layout for desktop, tablet, and mobile devices
* Smooth scrolling using Lenis
* GSAP-powered animations and transitions
* Component-based React architecture
* React Router navigation
* Responsive project gallery
* Clean and minimal UI inspired by the original K72 website

## Tech Stack

* React 19
* Vite
* Tailwind CSS v4
* GSAP
* @gsap/react
* Lenis
* React Router DOM

## Getting Started

Clone the repository:

```bash
git clone https://github.com/sakshamtnwr/k72-react-gsap.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
